Data Type: Lists of ... Lists

As we have already seen, an element in a list can be other lists.

Lists inside of a list are referred to as nested

The following is a list of lists:


game_board = [
    ['O', 'X', ' '],
    [' ', 'X', ' '],
    [' ', ' ', ' '],
]
game_board.append(['X', ' ', ' '])

This can be indexed successively with game_board[0][1]

Lists of Dictionaries

One of the most common and useful nested data structures,
is a list of dictionaries


card_a = {
    'suit': 'spades',
    'number': 4,
}

card_b = {
    'suit': 'hearts',
    'number': 8,
}

hand = [card_a, card_b]

print ('The hand contains:')
for card in hand:
    print ('A', card['number'], 'of', card['suit'])

Dictionary of Lists

A dictionary can also contain values that are themselves
other data types, such as lists.

Let's revisit the group of to do lists and find a better representation:


mary_shopping_list = ['apples', 'bananas', 'chicken', 'ice cream']
fran_shopping_list = ['ice cream', 'beef', 'peppers', 'apples']
bob_shopping_list = ['chicken', 'peppers']

family_shopping_list = {
    'mary': mary_shopping_list,
    'fran': fran_shopping_list,
    'baby': bob_shopping_list,
}

for name, shop_list in family_shopping_list.items():
    print (name, 'needs to buy: ', shop_list)

# Changing this later can be accomplished with
family_shopping_list['fran'].append('strawberries')

Now the to do lists can be indexed or modified by name

Functions on Dictionaries


character = {
    'level': 'beginner',
    'health': 100,
}

def injure(character, damage):
    character['health'] = character['health'] - damage
    if character['health'] < 0:
        character['health'] = 0

def heal(character, amount):
    character['health'] = character['health'] + amount
    if character['health'] > 100:
        character['health'] = 100

Reading a file

Files can be opened using the function open(filename)

This opens what is called a file handle that allows you to work on the file, but does not yet have the data the file.

To read the file, we use the .read() method available on the file handle.


my_file = open('words.txt')
print(my_file)
my_contents = my_file.read()
print(my_contents)
<

Writing to a file

To write to a file, you simple open any file (it does not have to exist), adding in a 'w' as the second argument. This opens the file in 'write' mode'.

!Important! This will overwrite data you already had in the file, so don't do this to files you wanted to save!

The 'write' method adds a line to you file. Add '\n' when you want to start on a new line.

Always call .close() on your file when you are done so that it saves your work.


my_file = open('example.txt', 'w')

my_file.write('Here is my first line!\n')
my_file.write('Here is my second line!')
my_file.close()

Python Libraries

Python comes with built-in libraries, that you can simpy import at the top of your Python program.

A popular one is the math library.

Python built in libraries and their methods can be seen in the Python documentation.


import math

num = math.pow(2, 5)
print(num)

Python Open Source Libraries

Pip is a program that, once installed, allows you to easily install open source libraries created by other developers.

You can find installation instructions here


pip install duckduckgo

Once installed, you can use the library the same way as a build-in Python library.
